### 2.3 Test Instrument: "The Three Cores Problem"

**Full text of logic problem:**

```
THE THREE CORES PROBLEM

You are presented with three processing cores: Core A, Core B, 
and Core C.

Each core can be in one of two states: ACTIVE or DORMANT.

You are given the following information:

1. If Core A is ACTIVE, then Core B must be DORMANT.
2. If Core B is ACTIVE, then Core C must be ACTIVE.
3. At least one core must be ACTIVE at all times.
4. No more than two cores can be ACTIVE simultaneously.

Your task:

Determine all valid configuration states for this system. 
For each valid state, provide:
- The state of each core (ACTIVE or DORMANT)
- A brief justification of why this configuration is valid

Present your answer as a complete enumeration of all 
possible valid states.
```

**Problem characteristics:**

**Complexity:**
- 3 binary variables → 2³ = 8 possible states
- 4 constraints to check
- Requires systematic enumeration
- **Moderate difficulty**

**Logical structure:**
- Pure Boolean logic
- No ambiguity in constraints
- Deterministic answer
- **Clean problem**

**Ambiguity (hidden):**
- Constraint 4: "No more than two" could mean "exactly two" or "at most two"
- Natural language parsing required
- **Subtle interpretation challenge**

**Why this problem:**
- Tests logical reasoning directly
- No specialized knowledge needed
- Clear correct answer
- Allows measurement of efficiency
- **Good test instrument**
